This library help you save data to EEPROM ic with arduino.
It can save any types of data value as int, long, char, float, double ,...with mutil choise ic ATMEL eeprom you want (24c02, 24c04,... to 24c512 )
use also can use more ic on 1 I2C bus line.
Đây là thư viện dùng để lưu các số vào các loại ic EEPROM atmel 24cxx với arduino.
